## Installation
1. Clone the repository:
```bash
    git clone https://github.com/pbelltech0/news-api-mcp.git
```

2. Install uv dependencies
```bash
    uv pip install -r pyproject.toml
```

3. Update .env file with your https://newsapi.org/ API KEY
